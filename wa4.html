The television show, Friends, aired for ten seasons. The show featured six friends, named Monica, Chandler, Ross, Joey, Phoebe, and Rachel.. Rachel talks too much in this show and is widely regarded as the most annoying character. This analysis looks at the amount of dialogue each friend had in each episode of the show. The hope in this analysis is to address the disparity in dialogue between the actors. The data was sourced from a github user, linked above. The data provided integer data based on how many "lines" the character had.
